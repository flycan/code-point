# linux中目录文件
```
|-bin 基础命令可执行文件
|-boot 启动引导目录
|-dev （device）设备文件 光盘 硬盘分区
|-etc  系统配置文件
|-home 普通用户的家目录
|-lib   库文件
|-lib64 库文件 存放共享库
|-lost+found 发生错误时存放文件目录
|-media 多媒体文件
|-mnt  挂载目录
|-opt （option）第三方软件安装目录
|-proc （process）虚拟目录 存放进程信息 系统信息等
|-root root用户家目录
|-sbin root可执行命令二进制文件
|-srv service
|-sys 虚拟目录
|-tmp 临时存放文件 回收站
|-usr 用户软件
|-var （variable）存放经常变更的数据 日志等
